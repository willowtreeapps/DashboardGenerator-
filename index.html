WTDashboardJSONGenerator

Instructions:

    Upload a JSON file to generate a completed grid, or create a grid from scratch.

    Enter column and row information to generate grid.

    Drag and drop to reorder.

    Select elements and click "Resize" button to resize selected cell to the input width and height.

    Delete cells to create empty spaces.

    Select a Job and Widget and select the cell to use the right textarea to edit the config JSON for the highlighted cell.

    Select "Generate JSON" to get JSON string once you are happy with the configuration.

    Press the "Copy Text" button at the very bottom to copy the generated JSON text to your keyboard.
